Kebijakan Privasi

Terakhir diperbarui: Januari 2025

🔒 Ringkasan singkat: Kami tidak mengumpulkan data pribadi kamu. API key tersimpan di browser kamu sendiri, bukan di server kami.

1. Data yang Kami Kumpulkan

Vidarix dirancang dengan prinsip privacy-first. Kami tidak mengumpulkan, menyimpan, atau memproses data pribadi identifiable. API key Anthropic yang kamu masukkan hanya tersimpan di localStorage browser kamu dan tidak pernah dikirim ke server kami.

2. Konten yang Kamu Input

Teks atau deskripsi video yang kamu masukkan ke dalam AI agent dikirim langsung dari browser kamu ke API Anthropic — tidak melewati server Vidarix. Artinya, kami tidak pernah melihat atau menyimpan konten inputan kamu.

3. Analytics & Cookies

Kami menggunakan analytics dasar (seperti jumlah pengunjung halaman) tanpa mengumpulkan informasi yang bisa mengidentifikasi kamu secara personal. Tidak ada cookie tracking pihak ketiga yang digunakan.

4. Pihak Ketiga

Satu-satunya pihak ketiga yang menerima data kamu adalah Anthropic (penyedia AI), sesuai kebijakan privasi mereka di anthropic.com/privacy. Kamu bertransaksi langsung dengan Anthropic menggunakan API key kamu sendiri.

5. Keamanan Data

Karena kami tidak menyimpan data kamu di server kami, risiko kebocoran data dari sisi kami sangat minimal. Namun tetap pastikan kamu menggunakan browser yang aman dan tidak mengakses Vidarix dari perangkat publik saat memasukkan API key.

6. Hak Kamu

Kamu bisa menghapus API key yang tersimpan kapan saja melalui pengaturan browser (clear localStorage). Untuk pertanyaan terkait privasi, hubungi kami di hello@vidarix.com
